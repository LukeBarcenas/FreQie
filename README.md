# FreQie
A web app for getting the frequencies of words in a txt file

#### Table of Contents
- [Setup](#setup)
- [Features](#features)

## Setup
Simply head to https://freqie.onrender.com and enjoy the features of FreQie!

(There may be some delays because of the hosting service)
